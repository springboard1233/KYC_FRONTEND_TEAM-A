
## ⚡ Getting Started

### 🎨 Frontend (React)
```bash
cd frontend
npm install
npm start
```
Runs the React app on http://localhost:3000

🖥️ Backend (Node.js API)
```bash
cd backend
npm install
npm start
```
Runs the backend API on http://localhost:5000

🔍 Text Extraction Service (Python OCR)
```bash
cd backend
python app.py
```
